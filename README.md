# 🌎 Rota Viva Piauí

Projeto desenvolvido para a seletiva **Piauí para o Mundo 2026**.

---

## 📌 Objetivo

Valorizar a cultura piauiense através da tecnologia, promovendo turismo cultural, economia criativa e inclusão digital.

---

## 💡 Sobre o Projeto

O Rota Viva Piauí é uma plataforma digital criada para conectar turistas, moradores e produtores culturais locais.

A proposta busca divulgar:
- Artesanato
- Culinária típica
- Turismo cultural
- Eventos regionais
- Experiências locais

---

## 🚀 Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- GitHub
